Detailed Description

template<typename StreamType>
class RF24Logger< StreamType >

A templated class for implementing a single logging object. Copied logging (using RF24Logger(RF24Logger*)) objects will re-use the original handler and level.

template<typename StreamType >
StreamType* RF24Logger< StreamType >::getLogger ( )

Returns the logging handler that was configured with RF24Logger(S*) or setHandler()

Example usage:

logging.getLogger() << "a string of text " << 0 << ':' << 1.0 << endl;

would ouput "a string of text 0:1.0" with a trailing line feed.
